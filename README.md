# Accessibility Demo Page

This repository contains a single **HTML page (`index.html`)** with intentionally included web accessibility issues.

## Purpose
The page is designed for learning and testing accessibility tools. Developers or students can run an accessibility audit and practice identifying and fixing common issues.

## File
- `index.html` – Demo page containing several accessibility violations.

## How to Use
1. Open `index.html` in a web browser.
2. Run an accessibility audit using tools such as **axe DevTools**.
3. Review the detected issues and apply accessibility fixes.

## Goal
Help developers understand common accessibility problems and how to remediate them using accessibility best practices.
